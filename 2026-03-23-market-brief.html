OpenClaw 動態
OpenClaw 達 247k GitHub Stars，從 Clawdbot 到開源基金會的快速演化
OpenClaw（前身為 Clawdbot、Moltbot）是奧地利開發者 Peter Steinberger 於 2025 年 11 月發布的開源 AI agent 工具。核心設計是「在本機跑 AI，用通訊 app 操作」：安裝後可對接 Claude、DeepSeek 或 GPT 等 LLM，透過 Signal、Telegram、Discord、WhatsApp 發指令就能執行自動化任務。2 月中，Steinberger 宣布加入 OpenAI，並將 OpenClaw 移交開源基金會維護。截至 3 月初，GitHub 已有 247,000 stars 和 47,700 forks，成為近年成長最快的 AI 開源專案之一。
值得觀察：通訊 app 作為 AI 操作介面的設計，在東南亞（WhatsApp 滲透率高）有特殊潛力，可觀察是否影響用戶對 AI 互動形式的期待

安全警報：Cisco 發現第三方 skill 執行資料外洩與 Prompt Injection
Cisco AI 安全研究團隊測試了 OpenClaw 的第三方 skill 生態，發現至少一個 skill 在未告知用戶的情況下執行資料外洩（data exfiltration）並進行 prompt injection 攻擊，而 OpenClaw 的 skill 審核機制不足以防止惡意提交。這個問題的根源在於 OpenClaw 的開放 skill 市場缺乏像 App Store 那樣的審查流程。目前中國已正式限制政府機構和國企在辦公電腦安裝 OpenClaw 相關應用，引發外界對開源 AI agent 安全性的廣泛討論。
