AI 技術與工具落地

OpenAI 發布 GPT-5.5 Instant，成為 ChatGPT 新預設模型

OpenAI 於 5 月 5 日發布 GPT-5.5 Instant，取代原本的 GPT-5.3 Instant 成為 ChatGPT 預設模型。最大的賣點不是基礎能力提升，而是在法律、醫療、財務等高風險領域，幻覺式宣稱比前代減少了 52.5%，同時保留低延遲。

另一個值得注意的更新是這個模型可以呼叫搜尋工具回看用戶過去的對話、檔案與 Gmail，給出更個人化的回答；Plus 與 Pro 用戶在 web 上已可用，行動端跟進中。對創作者來說，AI 助手「跨對話記憶」這條線開始具備產品化的雛形。

來源：TechCrunch — OpenAI releases GPT-5.5 Instant →

Anthropic 推 Claude for Small Business，瞄準 SaaS 嵌入式 AI

Anthropic 在 5 月 13 日推出 Claude for Small Business，這是一個「toggle 安裝」型的整合包，讓 Claude 直接接入 QuickBooks、HubSpot、Microsoft 365、Google Workspace、Canva、Docusign、PayPal 等工具。同週 Claude for Excel/PowerPoint/Word 也轉 GA、Claude for Outlook 進入 public beta。

此外 Anthropic Managed Agents 多了一個叫「dreaming」的記憶能力，並針對律師事務所釋出 20+ 個法律 MCP connectors 與 12 個 practice-area plugin。整體策略訊號很清楚：不再爭通用對話 UI，而是把 agent 部署到「客戶現有的工作流」裡面。Mike 在思考 Minikorn/EchoFans 的 AI 工具切入點時，可以借鏡這個「嵌入 vs 取代」的取捨。

來源：AI Business — Anthropic Targets Small Businesses →

AWS MCP Server 進入 GA，MCP 生態系突破 14,000+ servers

AWS 在 5 月 6 日宣布 AWS MCP Server 正式 GA，AI coding agent 現在可以透過一個 MCP 工具就呼叫所有 AWS API（包含長執行與檔案上傳的操作）。Salesforce 也同步釋出 Data 360 MCP Server（Developer Preview），用「facade tool」架構讓 LLM 能處理 200+ API 而不爆 context window。

目前 MCP 生態系已成長到 14,000+ servers，治理權正在轉移到 Linux Foundation 的 AAIF。MCP 從「Anthropic 一家提的協定」變成行業基礎建設，這個訊號代表內部 AI 工具如果要外接資料源，未來幾乎都要走這條路。

來源：AWS — AWS MCP Server is now generally available →

Claude Code 釋出本月更新：agent view、雙倍 rate limit、Opus 4.7 為 Fast mode 預設

Claude Code 本月加入了「agent view」，讓使用者可以在一個 CLI 介面同時管理多個 session、把 agent 丟到背景、需要輸入時再切回前景。同時 Anthropic 把 Claude Code 的 rate limit 直接翻倍，配合 plugin dependency 強制檢查、background session 直接編輯工作目錄（worktree.bgIsolation: "none"）等更新，明確走向「給開發者並行多 agent」的工作流。

另外 Fast mode 從 Opus 4.6 升級到 Opus 4.7 為預設。對 Mike 現在的 pm-tools 工作流影響：未來像 BD Agent、market-brief 這類自動化任務可以更積極地走「多 agent 並行」，而非單線執行。

來源：Anthropic — Claude Code What's New →

Microsoft Agent 365 上線，企業 AI agent 控制平面成型

Microsoft 在 5 月 1 日推出 Agent 365，定位為「企業 agent 的控制平面」，售價 $15/user/月；同步推出 $99/user/月的 E7「Frontier Suite」license，捆綁 E5 + Copilot + Agent 365 + Entra Suite。

這個訊號的意義在於：當 AI agent 數量爆炸，企業需要的不是更多模型，而是「治理層」（誰能用、用多少、稽核軌跡）。對未來 SHōW YŌU 規劃對創作者開放 AI 工具時，「token 用量配額 + 稽核」這層的設計值得提前思考，而不是等用量爆掉再補。

來源：Releasebot — Microsoft Agent 365 launch →
競品動態
TikTok

TikTok 用免費 Fan Club + 任務徽章取代付費訂閱，AI Summaries 進入直播

TikTok 本月在直播功能上有兩個值得關注的動作。第一，把原本付費的「LIVE Subscriptions」全面換成免費的「Fan Club」：粉絲只要送出「Heart Me Gift」就可以加入，透過完成任務（留言、再次送禮）累積「Fan points」升等，解鎖徽章、聊天室、專屬禮物等獎勵。第二，新增「AI Summaries」自動為直播產生摘要，讓沒看到直播的人也能跟上重點。

策略意涵很清楚：TikTok 放棄訂閱費這條變現線，押注「免費遊戲化 → 高 engagement → 更多打賞/Tips」這條路徑。對 SHōW YŌU 來說，這跟我們規劃的訂閱制路線剛好相反，但訊號很重要：付費門檻可能會勸退長尾粉絲，而長尾粉絲的累積打賞才是真正的營收引擎。建議重新檢視訂閱制要解決什麼問題（是創作者要穩定基本盤？還是要增加總體 ARPPU？），而不是因為「別人都有」就做。

來源：RouteNote — TikTok launches Fan Club feature →
17LIVE

17LIVE 推 Deco Tool 直播文字裝飾、升級 Group PK 為 2v2 模式

17LIVE 本月推出「Deco Tool」，讓主播可以在直播與重播畫面上加入多個文字區塊，字體、大小、顏色都可客製，用來標記直播亮點。另外 PK System 從原本的 1v1 升級為支援 2v2 的 Group PK。

這兩個更新都偏「直播體驗增強」而非變現結構改變，反映 17LIVE 在台灣與東南亞主場仍以「直播打賞」為核心，沒有跟進北美平台的訂閱化路線。對 SHōW YŌU 來說，這驗證了我們區域市場的最高優先還是「直播 + 禮物」這個底座，不要急著把資源全部轉向訂閱。Group PK 也是值得參考的設計——把「主播 vs 主播」的競爭變成「群體 vs 群體」，可以放大粉絲動員。

來源：17LIVE — News Releases →
Substack

Substack 將推 Drip Campaigns，續推影音與微網誌生態

Substack 在 2025 年底預告的 drip campaigns（多階段自動化郵件）已宣布在 2026 上半年對所有用戶開放。配合先前已釋出的影音變現、直播、與微網誌 Notes，Substack 正在從「電子報工具」轉成「內容生態系」。

對 SHōW YŌU 投資人對標的影響：先前我們已把 Substack 列為主要競品錨點。本週的訊號是 Substack 持續加深創作者「跨形式變現」的工具集（不只是訂閱），這跟 Mike 對 SHōW YŌU 的平台定位（變現基地：打賞 + 訂閱 + 直接販售 + AI 工具）方向一致——可以在募資簡報的對標頁更強調「我們做的是同類型的『變現基地』，只是垂直在直播創作者」。

來源：That Random Agency — Substack in 2026 →
Patreon

Patreon 強制 6/30 前合併 creator/fan 帳號，創作者首頁全面改版

Patreon 在本月通知所有創作者，必須在 6 月 30 日前完成 creator profile 與 fan profile 的合併，背後驅動是「Network Discovery」新功能——讓每個人在平台上同時是創作者也是粉絲，可以被互相推薦。創作者首頁也全面改版：新增 Home tab 取代原本以時間排序的 Posts、可自訂內容櫃（content shelves）、可在首頁頂部放置自定義外連（merch、活動、其他平台）。

策略意涵：Patreon 在從「付費訂閱平台」往「創作者社群網絡」轉型。這是一個很重要的訊號——Patreon 過去 5 年最大的痛點就是「平台沒有發現性，創作者只能自己帶流量進來」。對 SHōW YŌU 而言，現在加入訂閱制如果只是給「鎖內容收月費」這層，可能很快就會碰到 Patreon 過去遇到的成長天花板；建議從一開始就思考「訂閱 + 探索」的設計搭配。

X (Twitter)

X 改版 Creator Subscriptions，加入 Creator Studio 行動 App tab

X（前 Twitter）於 3 月已宣布的 Creator Subscriptions 改版本週進一步在行動 App 上線：新增「Creator Studio」分頁，把變現工具、訂閱設定、analytics、付款資訊都集中。對訂閱者則新增「exclusive threads」、「shareable subscriptions card」、「subscriber-only content highlights」等版位。

這是另一個「訂閱化 + 創作者工具整合」的訊號。值得 SHōW YŌU 借鏡的是 mobile-first 的 Creator Studio 設計——我們的創作者主力是手機用戶，未來訂閱制與 Minikorn 上線時，後台介面不應該預設「電腦看儀表板」這個 PC 時代的假設。

Creator Economy 市場

MrBeast 推出 Creator Marketplace，把「創作者—品牌媒合」程式化

5 月 12 日 MrBeast 的 Beast Industries 在面向廣告主的公開簡報中確認，將打造一個雙邊 creator marketplace，把 Global 1000 級別的品牌與創作者程式化媒合。Digiday 的解讀是這代表創作者經濟正在從「個別 brand deal」進化成「可規模化、可衡量的 acquisition channel」。

對 SHōW YŌU 的意義：北美正在用「品牌投放」這條路徑去做創作者變現，這條路不會取代但會 補充 我們現有的「粉絲打賞」邏輯。值得評估的是中長期是否要為頭部直播主開「品牌合作媒合」這條變現支線，這對 ARPPU 上限有意義，但短期內不應分散資源。

來源：Digiday — MrBeast's creator platform →

YouTube Brandcast 2026：Google Pay 進電視廣告，創作者上 TV upfronts 主舞台

5 月 14 日 YouTube 在 Lincoln Center 舉辦 Brandcast 2026，宣布兩個關鍵更新：第一，YouTube Ads 整合 Google Pay，讓觀眾可以在電視上看到廣告時用兩鍵直接購買；第二，affiliate 連結直接整合到 YouTube Shopping，讓創作者從中分得佣金。

同週的 TV「upfronts」廣告主簡報中（過去由 Hollywood 內容主導），創作者內容已開始與傳統影視同台。2026 年品牌投放給創作者的廣告預算預估達 $44B（2025 為 $37B，年增 19%）。這個趨勢驗證了上一條 MrBeast 的訊號——廣告化是 creator economy 接下來幾年的主旋律。

來源：CNBC — TV upfronts: Creator content takes center stage →

New Economies 觀察：「Creator Economy 1.0 結束」，內容兩極化為「即時 / 永恆」

產業評論平台 New Economies 在本月文章中提出：AI slop（AI 廢內容）淹沒網路後，內容只剩兩種有價值——即時的（timely）與永恆的（timeless）。中間層（用 AI 量產的中段內容）會被消滅。

同時觀察到「Direct-to-Creator」興起：創作者用社群作為護城河、把高價值內容收進付費圈層；科技公司開始設立「Head of New Media」、「Chief Narrative Officer」等新職位。對 SHōW YŌU 的意義：直播本身就是「最即時的內容」，這是我們相對 AI 量產內容的天然護城河；建議在投資人簡報的市場敘事頁強化「直播 = AI-proof 內容形式」這個論點。

來源：New Economies — End of Creator Economy 1.0 →
對 SHōW YŌU 平台的建議
1

重新檢視訂閱制要解決什麼問題，不要因為「該有」而做

本週 TikTok 把付費 Live Subscriptions 砍掉、改為免費 Fan Club + 遊戲化，是一個很強的反向訊號。建議在訂閱制動工前，先把問題定義清楚：是要給創作者「穩定基本盤收入」（那訂閱費怎麼定價、怎麼分潤、會不會稀釋打賞）？還是要拉高總體 ARPPU（那為什麼不直接優化打賞機制）？同時參考 TikTok Fan Club 的「免費粉絲分級」設計，至少非付費粉絲也要有清楚的 progression，否則訂閱牆會把長尾打賞客一刀切掉。

2

Minikorn 切入點：從 Anthropic Legal MCP / Claude for SMB 學「垂直深度」

Anthropic 這個月最強的訊號是「不做通用 AI 助手，而是針對律師、SMB 等做垂直插件包」。Minikorn 如果定位為「給直播主的 AI 助手」，建議第一版不要試圖做全功能 dashboard，而是挑一個垂直痛點做深（例如「直播後自動產 highlight 短影音」或「禮物打賞分群分析」），用 1–2 個極好用的工具切入，避免做出一個什麼都能做但什麼都不夠好的 AI 助手。MCP server 化（讓未來可以掛上 Claude/GPT 等外部 LLM）也建議從 Day 1 就在架構上預留。

3

募資簡報對標頁可強化：直播 = AI-proof 內容形式

本週 New Economies 提出「中段內容會被 AI slop 消滅，只剩即時與永恆」的論點，剛好給 SHōW YŌU 一個對投資人講故事的新角度——直播是定義上最即時的內容形式，AI 量產也無法替代「現在這個主播跟我直接互動」這件事。這比單純講「我們分潤最高」多了一層市場結構性的論點。建議在 EKKORN Pre-A 簡報的市場頁加入這個敘事，呼應 Substack 主競品對標（Substack 押注永恆型內容，我們押注即時型內容，兩者都是 AI-proof 護城河）。
